This directory contains anionic and cationic itp files to run "water in salt" electrolytes
simulations using GROMACS 5.0.7 with the OPLS-2009IL force field, the optimized Lennard-Jones Parameters for cations, and the SPC/E model for water.

See papers:
Doherty, B.; Zhong, X.; Gathiaka, S.; Li, B.; Acevedo, O. "Revisiting OPLS Force Field Parameters for Ionic Liquid Simulations."
J. Chem. Theory Comput., 2017, 13, 6131-6145. doi:10.1021/acs.jctc.7b00520

Jensen, K. P.; Jorgensen, W. L. "Halide, Ammonium, and Alkali Metal Ion Parameters for Modeling Aqueous Solutions. "
Journal of Chemical Theory And Computation 2006, 2, 1499-509. doi:10.1021/ct600252r

Berendsen H J C, Grigera J R, Straatsma T P. "The Missing Term in Effective Pair Potentials. "
The Journal of Physical Chemistry, 1987, 91(24): 6269-6271. doi:10.1021/j100308a038
